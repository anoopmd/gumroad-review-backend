# Slick Review Backend Services
Backend Services for the Slick Review App.

## Dependencies
* Npm >= v6
* NodeJs >= v10
* MySQL >= 5.7

## Install
```bash
sudo npm install
```

## Workflow
```bash
# build
npm run build

# run
npm start
```

## Frontend
The frontend services repo is located [here](https://github.com/anoopmd/slick-review-v1-app)
